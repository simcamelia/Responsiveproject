How this page works

  • Styled with Tailwind; brand colors set using CSS variables.
  • Product cards carry a JSON payload in data-product.
  • “View” opens a modal with image/description/price.
  • “Add” updates the cart sidebar; “Buy Now” opens a quick checkout message.
  • Facebook Page embedded with the official Page Plugin iframe.
